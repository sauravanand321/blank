Detailed History of Cricket World Cup

Here's a look at the winners and key stats from every edition of the ICC Men's ODI World Cup since its beginning in 1975.

1975

Winner: West Indies

Finalist: Australia

Semifinalists: England, New Zealand

Highest Run Scorer: Glenn Turner (NZ) - 333 runs

Highest Wicket Taker: Gary Gilmour (AUS) - 11 wickets

1979

Winner: West Indies

Finalist: England

Semifinalists: Pakistan, New Zealand

Highest Run Scorer: Gordon Greenidge (WI) - 253 runs

Highest Wicket Taker: Mike Hendrick (ENG) - 10 wickets

1983

Winner: India

Finalist: West Indies

Semifinalists: England, Pakistan

Highest Run Scorer: David Gower (ENG) - 384 runs

Highest Wicket Taker: Roger Binny (IND) - 18 wickets

1987

Winner: Australia

Finalist: England

Semifinalists: Pakistan, India

Highest Run Scorer: Graham Gooch (ENG) - 471 runs

Highest Wicket Taker: Craig McDermott (AUS) - 18 wickets

1992

Winner: Pakistan

Finalist: England

Semifinalists: New Zealand, South Africa

Highest Run Scorer: Martin Crowe (NZ) - 456 runs

Highest Wicket Taker: Wasim Akram (PAK) - 18 wickets

1996

Winner: Sri Lanka

Finalist: Australia

Semifinalists: India, West Indies

Highest Run Scorer: Sachin Tendulkar (IND) - 523 runs

Highest Wicket Taker: Anil Kumble (IND) - 15 wickets

1999

Winner: Australia

Finalist: Pakistan

Semifinalists: South Africa, New Zealand

Highest Run Scorer: Rahul Dravid (IND) - 461 runs

Highest Wicket Taker: Geoff Allott (NZ) & Shane Warne (AUS) - 20 wickets each

2003

Winner: Australia

Finalist: India

Semifinalists: Kenya, Sri Lanka

Highest Run Scorer: Sachin Tendulkar (IND) - 673 runs

Highest Wicket Taker: Chaminda Vaas (SL) - 23 wickets

2007

Winner: Australia

Finalist: Sri Lanka

Semifinalists: New Zealand, South Africa

Highest Run Scorer: Matthew Hayden (AUS) - 659 runs

Highest Wicket Taker: Glenn McGrath (AUS) - 26 wickets

2011

Winner: India

Finalist: Sri Lanka

Semifinalists: Pakistan, New Zealand

Highest Run Scorer: Tillakaratne Dilshan (SL) - 500 runs

Highest Wicket Taker: Shahid Afridi (PAK) & Zaheer Khan (IND) - 21 wickets each

2015

Winner: Australia

Finalist: New Zealand

Semifinalists: India, South Africa

Highest Run Scorer: Martin Guptill (NZ) - 547 runs

Highest Wicket Taker: Mitchell Starc (AUS) & Trent Boult (NZ) - 22 wickets each

2019

Winner: England

Finalist: New Zealand

Semifinalists: India, Australia

Highest Run Scorer: Rohit Sharma (IND) - 648 runs

Highest Wicket Taker: Mitchell Starc (AUS) - 27 wickets

2023

Winner: Australia

Finalist: India

Semifinalists: New Zealand, South Africa

Highest Run Scorer: Virat Kohli (IND) - 765 runs

Highest Wicket Taker: Mohammed Shami (IND) - 24 wickets
